# 第五章摘要：血路·突围

## 概括
林零和铁牛联手突袭昆仑重工前哨站。铁牛正面牵制火力，林零从通风管道潜入服务器室删除铁牛产权记录，意外发现易筋篇线索。撤退时遭遇昆仑执行官"灭世金刚"，林零以洗髓剑意刺穿其装甲间隙，铁牛以残躯挡住灭世金刚掩护林零逃脱。两人最终在废墟区汇合，决定返回幽冥层。

## 关键事件
1. 两人分工：铁牛正面吸引铁卫+狙击手火力，林零通风管道渗透
2. 林零利用洗髓篇控制体温通过红外传感器——戒断期副作用变优势
3. 成功删除铁牛产权记录（KN-TB-3357），远程回收协议进入待定状态
4. 发现昆仑数据库加密文件"铁血心印·易筋篇·峨眉分区"——易筋篇在峨眉生物手中
5. 遭遇灭世金刚（昆仑白芯级执行官），林零找到装甲间隙以洗髓剑意刺入
6. 铁牛以残躯冲锋撞开灭世金刚，掩护林零逃脱，自己跳窗突围

## 角色状态变化
- 林零：右肋被灭世金刚冲击波擦伤，首次产生"保护什么人"的念头
- 铁牛：左臂断裂，右腿废了，胸甲凹陷，铁布衫碎裂，但远程协议已暂停
- 两人关系：初识信任→生死之交（铁牛为林零挡灭世金刚，林零扛铁牛走回幽冥层）

## 新伏笔
1. 易筋篇在峨眉生物手中——下一阶段目标
2. 灭世金刚没死——昆仑重工的威胁仍在
3. 林零用洗髓篇控制体温通过传感器——戒断期副作用可能是新能力的前兆
4. 林零第一次主动扛着别人走——独行者心态开始改变

## 精彩对话
- 铁牛（对灭世金刚）："三年没见了。你还是喜欢在走廊里堵人。"
- 林零内心："那个念头很轻，像洗髓篇在脊椎里留下的温热——只是因为他想扛着这个人走。"

## 情感节拍
潜入的紧张→发现易筋篇线索的震撼→遭遇灭世金刚的压迫→铁牛挡刀的悲壮→并肩而行的温暖
