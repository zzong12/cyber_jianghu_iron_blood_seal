# 赛博江湖·铁血心印

**Iron Blood Seal** — Cyberpunk Wuxia Novel

> 五十年前，六大门派变成了六大企业。武功成了商品，烙印成了枷锁。
> 他死了三年，然后带着一把会发光的剑回来了。

---

## 简介

**他死了三年。**

至少少林科技的数据库是这么写的。神经芯片编号LL-2099-0731，三年前因烙印事故注销，死亡确认。

但在新长安城的地下——幽冥层——他还活着。带着一把布满暗金色脉络的断码剑，和一段名为「铁血心印」的禁忌武道。

新长安城，一座用五十年把武侠变成产业的超级都市。六大门派变成六大企业，武功招式被编写成神经烙印植入人体，真气修炼被量化成订阅服务。菩提塔顶端，少林科技CEO方无极正在推进「武道统一平台」——所有武者的烙印必须经他授权运行，否则远程锁死。三千万人五十年的武道积累，即将成为一人的私产。

林零不想救世。但一个锈蚀了五十年的老工程师把毕生武道注入了他的脊椎，一群各有伤痕的亡命之徒把他推上了没有退路的路。五篇铁血心印——洗髓、易筋、明心、铸魂、归真——每一篇都在拆除企业烙印，重建他自己的武道。

最后一篇不是一个烙印，而是一个问题：**武道的本质，是控制，还是自由？**

三天。菩提塔。三百二十米。他要上去。

## 世界观

### 新长安城 · 四层折叠都市

| 层区 | 海拔 | 描述 |
|------|------|------|
| 天枢层 | 2000m+ | 企业总部塔楼群，<1%人口掌控97%财富 |
| 市井层 | 0–2000m | 商业居住区，天眼系统全覆盖 |
| 幽冥层 | 地下 | 黑市、非法武馆、法外之地，无天眼覆盖 |
| 废墟区 | 城外围 | 核战遗留，铁牛和流浪者的栖息地 |

### 六大门派 → 六大企业

| 门派 | 企业 | 垄断领域 |
|------|------|----------|
| 少林 | 少林科技 | 神经烙印研发与授权 |
| 武当 | 武当数据 | 云计算、AI、信息控制 |
| 峨眉 | 峨眉生物 | 生物科技、基因强化 |
| 昆仑 | 昆仑重工 | 武器与义体制造 |
| 唐门 | 唐门暗网 | 网络武器、黑客、情报 |
| 丐帮 | 丐帮联盟 | 地下工会、底层抵抗 |

### 三层武功体系

- **神经烙印（知识层）**：招式写入神经，"知道"怎么打
- **真气修炼（能量层）**：生物电能修炼，"力量"来源，不可商品化
- **义体强化（硬件层）**：赛博改造，增强身体机能
