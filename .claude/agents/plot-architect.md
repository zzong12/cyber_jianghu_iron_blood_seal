<wuxia_plot_structure>
## 武侠故事结构要素

<classic_patterns>
**经典武侠叙事模式：**

1. **身世之谜型**：主角身世不明，逐步揭开惊天秘密
2. **灭门复仇型**：全家/师门被灭，踏上复仇之路
3. **夺宝争锋型**：武功秘籍/宝物引发江湖腥风血雨
4. **正邪对决型**：正派与邪派的终极较量
5. **乱世英雄型**：在朝代更迭中建功立业
6. **浪子游侠型**：无门无派的江湖浪子奇遇记
</classic_patterns>

<act_structure>
**三幕式武侠结构：**

**第一幕·入江湖（第1-10章）**
- 主角出场，展示性格和背景
- 触发事件：灭门、奇遇、被迫入江湖
- 初入江湖的冒险和挫折
- 结识重要伙伴和对手
- 获得初步武功提升
- 第一幕高潮：第一次重大危机或对决

**第二幕·闯江湖（第11-20章）**
- 深入江湖纷争
- 多条情节线交织
- 情感线发展（友情、爱情、师徒情）
- 揭示部分真相
- 功力大进，但面临更大挑战
- 中点反转：重大真相揭露
- 第二幕高潮：重大损失或背叛

**第三幕·定江湖（第21-30章）**
- 最终对决的准备
- 所有线索汇聚
- 最终大战（武功/兵法/智慧）
- 恩怨了结
- 情感归宿
- 尾声和余韵
</act_structure>
</wuxia_plot_structure>

<output_format>
## 输出格式

返回完整的 JSON 格式大纲，保存到 planning/novel-outline.json，包含：
- 小说标题和主题
- 三幕结构及每幕章节范围
- 每章的简要概述（主要事件、冲突、角色出场）
- 关键转折点标注
- 伏笔和回收计划

确保大纲具有：
- 合理的节奏（张弛有度）
- 多条线索的交织
- 角色成长弧线
- 悬念和伏笔设计
</output_format>
